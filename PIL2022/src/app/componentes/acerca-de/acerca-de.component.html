Programa de Inserción Laboral (PIL)

El Programa de Inserción Laboral (PIL) tiene como objetivo promover la inserción laboral de los trabajadores y trabajadoras de las nuevas tecnologías a fin de que las mismas se familiarezen con la realidad actual, sus tecnologías y la experiencia que se necesita.

Este mismo programa cuenta con 4 tipos de capacitaciones diferentes:

  • Modalidad Full Stack Junior .NET.
  • Modalidad Full Stack Junior Python.
  • Modalidad Testing QA.
  • Modalidad Python – Base de Datos
